Analysis Summary: Latitude vs X

Welcome to my Latitude vs X Analysis Dashboard. The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this weather analysis, i pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

I assembled this dataset with pandas using jupyter notebook. I used matplotlob to plot various scatterplots with different aspect of the weather vs.latitude. Some of the factors that i considered during this analysis included: temperature, cloudiness, wind speed, and humidity. This site provides some basic information of this analysis like the data source and the visualizations that was created. I will also give descriptions of any trend and correlations witnessed.
